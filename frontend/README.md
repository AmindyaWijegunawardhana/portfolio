# Portfolio Frontend

Modern React frontend for the futuristic portfolio website with Tailwind CSS and Framer Motion animations.

## 🎨 Components

- **Navigation**: Fixed top navigation with smooth transitions
- **Hero**: Full-screen landing with animated background
- **About**: Personal story with timeline
- **Projects**: Interactive project showcase
- **Case Study**: Detailed modal for project information
- **Skills**: Dynamic skill display with progress bars
- **Experience**: Timeline of career progression
- **Contact**: Email submission form
- **Footer**: Minimal footer with links
- **AI Chat**: Floating assistant bubble

## 📦 Installation

```bash
npm install
```

## 🚀 Running

```bash
npm start
```

The app will open at `http://localhost:3000`

## 🏗️ Building

```bash
npm run build
```

## 🎨 Customization

### Theme Colors
Edit `tailwind.config.js` to change neon colors and spacing

### Content
Update `src/data/projectsData.js` with your projects, skills, and experience

### API Integration
Update API calls in contact form to match your backend URL

## 📱 Features

- Dark/Light mode toggle
- Responsive design
- Smooth animations and transitions
- Interactive micro-interactions
- Glassmorphic UI elements
- AI chat assistant
- Smooth scroll behavior

## 🔧 Technologies

- React 18
- Tailwind CSS
- Framer Motion
- React Icons
- Axios

## 🚀 Deployment

Deploy to Vercel, Netlify, or any static hosting:

```bash
npm run build
```

Push the `build/` folder to your hosting service.
