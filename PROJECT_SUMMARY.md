## 🚀 Getting Started (3 Steps)

### 1. Install Dependencies

```bash
cd portfolio
npm run install-all
```

### 2. Start MongoDB

```bash
mongod
```

### 3. Run Development Server

```bash
npm run dev
```

**Result**: Frontend at http://localhost:3000, Backend at http://localhost:5000

## 🎯 Customization Quick Tips

1. **Update Portfolio Content**
   - Edit: `frontend/src/data/projectsData.js`
   - Add your projects, skills, experience

2. **Change Colors**
   - Edit: `frontend/tailwind.config.js`
   - Update neon color values

3. **Add Images**
   - Replace placeholders with your images
   - Store in `frontend/public/` or CDN

4. **Connect Email**
   - Update API endpoint in Contact component
   - Set up backend email service

5. **Deploy**
   - Frontend: Vercel, Netlify
   - Backend: Railway, Render, Heroku
